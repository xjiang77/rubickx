# Claude Code 课程简介

## 这门课是什么

这是一门面向中文读者的 `Claude Code` 课程，目标不是单纯介绍 CLI 命令，而是帮助学习者理解：

- Claude Code 是什么
- 它为什么有效
- 它在真实代码库中如何工作
- 应该如何正确使用它
- 它的限制、风险和 guardrails 在哪里

## 目标受众

这门课主要面向：
- AI-native builders
- vibe coders
- 使用 AI coding tools 但并不完全理解底层机制的人
- 希望更好地 steer、debug、review AI coding workflow 的人

## 学完后应获得什么

学完这门课，学习者应能够：
- 理解 Claude Code 的基本工作模式
- 在代码库中更好地使用 Claude Code 做探索、实现和修改
- 识别 Claude Code 擅长与不擅长的任务
- 为 Claude Code 配置更合理的 guardrails
- 把 Claude Code 放进更大的 Dev Squad / review workflow 里

## 为什么这门课适合放在 rubickx

`rubickx` 的长期方向不是一门课，而是课程系统。

Claude Code 作为第一门课，适合作为模板，因为它天然连接：
- terminal-native coding workflow
- codebase understanding
- tool use
- agent loop
- review / git / guardrails

它可以为后续课程（如 Codex、OpenClaw、Agent Teams 等）提供一套可复用的表达结构。
