# Claude Code 课程大纲

## 课程目标

让学习者从“会用 Claude Code”进化到“理解 Claude Code 如何工作、何时可靠、如何把它接入真实开发系统”。

## 模块设计

### 模块 1：Claude Code 是什么
回答：
- 它是什么类型的工具
- 它和普通 chat、IDE 补全、脚本助手有什么区别
- 为什么它适合 terminal-native coding workflow

### 模块 2：Claude Code 的基本工作流
回答：
- 如何进入一个 repo
- 如何理解代码库
- 如何读、改、写、执行
- 如何处理 routine tasks

### 模块 3：Claude Code 的核心机制
回答：
- agent loop 是什么
- tool use 如何发生
- 上下文如何被利用
- 为什么 prompt 和工作区边界很重要

### 模块 4：Claude Code 在真实项目中的使用方式
回答：
- bug fix
- refactor
- docs / scripts
- exploration
- git workflow

### 模块 5：Claude Code 的限制、风险与 guardrails
回答：
- 什么时候容易出错
- 什么时候不能完全信任它
- 为什么需要 reviewer
- 为什么需要 worktree / tests / human gate

### 模块 6：Claude Code 在 Dev Squad 中的位置
回答：
- 它在一个多 agent 开发团队里扮演什么角色
- 如何与 Codex / reviewer / tech-lead 配合
- 如何从 solo tool 变成系统组件
